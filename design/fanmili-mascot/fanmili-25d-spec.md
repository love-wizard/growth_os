# 饭米粒 2.5D 形象方向

## 设计读法
饭米粒不是玩具吉祥物，而是亲子陪伴产品里的温和记录伙伴。新版保持二维米粒的轻量识别，同时增加轻微体积、稳定道具和可扩展动作。

## 固定识别点
- 主体：短胖米粒，顶部轻微不对称，轮廓比旧版更有记忆点。
- 表情：两颗深棕小眼睛、小微笑、低饱和腮红。
- 品牌锚点：低饱和绿色叶子书签，代表成长记录；默认款由右手轻轻拿着，不贴在身体上，不使用强荧光绿。
- 材质：2.5D 柔和渐变，不做写实 3D，不做塑料高光。

## 使用建议
- Tab 栏：继续用现有 CSS 二维版本，保持极小尺寸稳定。
- 首页/AI 教练/首次引导：可评估使用 `fanmili-25d-default.png` 的小尺寸版本。
- 成长记录空状态：使用 `fanmili-25d-record.png`。
- 月报/年报入口或空状态：使用 `fanmili-25d-report.png`。

## 禁止项
不要大眼睛、动物耳朵、衣服化、强黑色投影、复杂文字、写实 3D、过亮绿色。
